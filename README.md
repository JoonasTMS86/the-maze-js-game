The Maze, a free and open source JavaScript puzzle game with a level editor by Joonas Lindberg.

2026 Joonas Lindberg

Try out the user made levels or make your own levels with the level editor of "The Maze"!

The goal on each level depends on what the user has defined to be the goal on that level.
If there are colored balls on the level, then the goal is to get rid of all the balls by pushing them against other balls of the same color.
If there are 'X' spots on the level, then the goal is to move the crates onto those spots.

There can also be doors which can only be opened with a key.

There might be some lava on some levels. Lava can only be crossed by dropping a boulder into the lava.

There might be some fragile walls on some levels. Those can be blown up with a bomb.

Some levels can have electric gates which block your path if they're turned on.
Such a gate can be toggled on or off by pressing a button that affects that particular gate.

Controls:

Use the Arrow Keys to move the main character around

Level Editor:

To place the currently selected tile, just click the game area with the left mouse button.

To select the previous tile, press the Z key.

To select the next tile, press the X key.

To set the player starting position to particular tile, move mouse cursor over that tile and press the C key.

To load a user made level, press the L key - press Esc to cancel.

To save your level, press the S key - press Esc to cancel.

To edit the properties of a gate or button, move the mouse cursor over the gate / button, then press the Enter key. Press Enter again to leave the window.
